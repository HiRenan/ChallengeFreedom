# ChallengeFreedom

Projeto de teste técnico para construir um assistente de suporte com:

- `Dify Studio` para o fluxo conversacional
- `Knowledge Base` para responder dúvidas com base em documentação
- backend próprio em `Python + FastAPI`
- consulta de ticket por ID
- criação de ticket com validação e coleta de dados faltantes

## Status atual

O repositório está em fase inicial de planejamento.

Arquivos principais no momento:

- `AGENTS.md`: regras e decisões do projeto
- `plano.md`: breakdown completo, tijolo por tijolo

## Objetivo da entrega

Entregar uma solução que consiga:

- responder perguntas com base em documentação
- consultar tickets existentes
- abrir novos tickets
- pedir informações faltantes antes de criar um ticket
