# FAQ de Suporte

Esta base responde dúvidas comuns antes da abertura de um ticket.

## O que este suporte atende

O suporte atende dúvidas operacionais e incidentes relacionados a acesso, sistemas internos, e-mail corporativo, VPN e permissões básicas.

## Quando devo abrir um ticket

Abra um ticket quando o problema exigir análise, correção, acompanhamento ou registro formal do atendimento.

## Quando não preciso abrir um ticket

Se a dúvida for apenas sobre procedimento, política de atendimento ou informação geral, primeiro consulte a documentação de suporte.

## O que preciso informar para abrir um ticket

Você deve informar:

- nome do solicitante
- e-mail para contato
- assunto do problema
- descrição clara do ocorrido

## Como consultar um ticket existente

Para consultar um ticket, informe o ID completo no formato `TKT-<número>`.

## O que acontece se eu enviar poucas informações

Se faltarem dados mínimos, o assistente pedirá as informações ausentes antes de registrar o ticket.

## O suporte responde dúvidas sobre status?

Sim. Você pode consultar o ticket pelo ID para verificar se ele foi localizado e qual status está disponível no momento.

## Posso abrir mais de um ticket para o mesmo problema

Não é o ideal. Se já existir um ticket para o mesmo incidente, prefira consultá-lo antes de criar outro.
